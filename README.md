# Wilted-Chess-Engine
A very *wilted* chess engine.


# Features

Basic Iterative Deepening Search

Quiescent Search at Terminal Nodes

Soft/Hard Time Management

Basic Zobrist Hashing (Threefold Detecting)

Randomly Generated PSQTs
